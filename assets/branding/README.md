# Branding CuadernoPro

Colocar en esta carpeta los recursos de marca oficiales del proyecto:

- `cuadernopro.ico`: icono Windows para el ejecutable, el instalador y los
  accesos directos.
- `cuadernopro.png`: icono/favicon de Streamlit y usos graficos ligeros.

Tamanos recomendados:

- `.ico`: archivo multi-resolucion con 16x16, 32x32, 48x48 y 256x256 px.
- `.png`: imagen cuadrada con fondo transparente, preferiblemente 512x512 px
  o 256x256 px.

No usar imagenes, iconos ni logotipos con derechos de terceros si no existe una
licencia clara y compatible con el proyecto.

El logo y la marca CuadernoPro estan controlados por el proyecto para evitar
confusion con versiones modificadas o no oficiales.
