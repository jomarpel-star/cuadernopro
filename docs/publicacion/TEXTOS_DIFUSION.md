## E) Descripción larga

CuadernoPro es una aplicación libre para gestionar el cuaderno agrícola de una
explotación. Está pensada para agricultores que quieren una herramienta local,
sin cuotas, sin activación y sin depender de una suscripción para conservar sus
datos.

Incluye gestión de explotación, campañas, parcelas, cultivos, tratamientos,
fertilización, prácticas culturales, cosecha, contabilidad, mapas, PDF oficial
del cuaderno, revisión SIEX, exportación Excel asistida, importación de
catálogos SIEX y copias de seguridad con documentos adjuntos.

El instalador Windows permite usar CuadernoPro sin Docker, WSL, Python ni
terminal. Los datos se guardan en el equipo del usuario, en una base SQLite
local, y no se suben automáticamente a ningún servidor.

CuadernoPro se distribuye como software libre bajo GNU GPL v3 o posterior. El
programa completo es gratuito para agricultores. Se podrán ofrecer servicios
opcionales de instalación, formación, soporte o adaptación, pero la aplicación
seguirá siendo libre.
