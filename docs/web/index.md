# CuadernoPro

El cuaderno agrícola libre para que ningún agricultor se quede sin herramienta.

CuadernoPro es una aplicación gratuita para agricultores, sin cuotas ni
activación, que funciona en local y guarda los datos en el ordenador del
usuario.

[Descargar CuadernoPro](descargar.md)

## Beneficios

- Gestiona explotación, campañas, parcelas, cultivos y actuaciones desde una
  sola herramienta.
- Trabaja con base SQLite local: los datos son del usuario.
- Instalable en Windows sin Docker, WSL ni terminal.
- Permite generar PDF del cuaderno, revisar información SIEX y preparar Excel
  asistido.
- Incluye copias de seguridad con documentos adjuntos.

## Software libre

CuadernoPro es software libre bajo GNU GPL v3 o posterior. El programa completo
es gratuito para agricultores y no tiene licencias anuales obligatorias.

Puede haber servicios opcionales de instalación, formación, soporte o
adaptación, pero la aplicación completa sigue siendo libre.

## Datos locales

CuadernoPro guarda la base y documentos en el equipo del usuario. No sube datos
automáticamente a servidores externos.

En Windows instalado, los datos se guardan en:

```text
Documentos\CuadernoPro
```

## Módulos principales

- Explotación y campañas.
- Parcelas y SIGPAC.
- Cultivos y cálculo de árboles.
- Tratamientos fitosanitarios.
- Fertilización.
- Prácticas culturales.
- Cosecha multicultivo.
- Contabilidad.
- Mapas.
- PDF oficial del cuaderno.
- Revisión y exportación SIEX asistida.
- Importación de catálogos SIEX.
- Backup y restauración.
