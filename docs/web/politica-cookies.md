# Politica de cookies

Fecha de ultima actualizacion: 08/07/2026

CuadernoPro no utiliza cookies propias de analisis, publicidad o seguimiento en
esta web.

Esta web publica es una web estatica destinada a informar sobre el proyecto
CuadernoPro y facilitar la descarga desde el repositorio oficial de GitHub.

No se utilizan cookies para:

- analizar habitos de navegacion;
- crear perfiles de usuario;
- mostrar publicidad personalizada;
- realizar seguimiento entre sitios;
- medir conversiones comerciales.

## Cookies tecnicas

La web puede requerir cookies tecnicas o mecanismos equivalentes estrictamente
necesarios para la seguridad, disponibilidad o funcionamiento basico del
servicio, en su caso proporcionados por el alojamiento web. Estas cookies, si
existieran, no se usan para seguimiento ni publicidad.

## Servicios de terceros

Al pulsar enlaces externos, como GitHub, el usuario abandona la web de
CuadernoPro y pasa a navegar por sitios de terceros. Esos sitios pueden aplicar
sus propias politicas de cookies y privacidad.

## Cambios futuros

Si en el futuro CuadernoPro incorpora analitica, videos embebidos, formularios
externos u otros servicios que requieran cookies no tecnicas, se actualizara
esta politica y se solicitara el consentimiento cuando corresponda.

## Contacto

Para dudas sobre esta politica, utilizar el canal publico de soporte indicado en
la pagina de soporte de CuadernoPro.
