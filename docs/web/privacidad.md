# Privacidad

CuadernoPro funciona en local y guarda los datos en el equipo del usuario.

## Datos locales

En Windows instalado, los datos se guardan en:

```text
Documentos\CuadernoPro
```

En instalaciones Docker o de desarrollo, la carpeta habitual es:

```text
runtime/
```

## Sin subida automática

CuadernoPro no sube datos automáticamente a servidores externos. La base SQLite,
documentos, copias de seguridad y exportaciones permanecen bajo control del
usuario.

## Cookies

La web pública de CuadernoPro no usa cookies propias de análisis, publicidad o
seguimiento. La política de cookies está documentada en
`docs/web/politica-cookies.md` y en la página pública
`web_publica/politica-cookies.html`.

## Control del usuario

El usuario decide qué datos introduce, qué documentos adjunta, qué copias de
seguridad conserva y que archivos comparte con terceros.

## Copias de seguridad

El usuario es responsable de conservar copias de seguridad suficientes. Es
recomendable guardar copias en una ubicación distinta del equipo principal.
