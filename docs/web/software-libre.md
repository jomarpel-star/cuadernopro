# Software libre

CuadernoPro es software libre bajo GNU General Public License versión 3 o
posterior (`GPL-3.0-or-later`).

## Gratuito para agricultores

El programa completo es gratuito para agricultores:

- sin activación;
- sin cuotas;
- sin demo capada;
- sin límites artificiales de parcelas;
- sin licencias anuales obligatorias.

## Servicios opcionales

Se pueden ofrecer servicios opcionales de pago:

- instalación;
- puesta en marcha;
- formación;
- soporte prioritario;
- importación de datos;
- adaptaciones personalizadas.

Estos servicios no cambian la libertad de uso del programa.

## Marca CuadernoPro

La marca CuadernoPro está controlada para evitar confusión con versiones no
oficiales, modificaciones no autorizadas o servicios que puedan presentarse como
oficiales sin serlo.

Consulta `TRADEMARKS.md` para más detalle.
