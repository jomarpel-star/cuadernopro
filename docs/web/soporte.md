# Soporte

## Soporte comunitario

El soporte comunitario se canaliza desde el repositorio público mediante issues,
documentación y contribuciones.

## Reportar errores

Al reportar un error, incluye:

- versión de CuadernoPro;
- sistema operativo;
- pasos para reproducir el problema;
- mensaje de error si aparece;
- captura si ayuda a entender el fallo.

No adjuntes bases de datos, facturas, recetas ni documentos con datos reales en
issues públicos.

## Servicios opcionales

Pueden ofrecerse servicios opcionales de instalación, puesta en marcha,
formación, soporte prioritario, importación de datos y adaptaciones.
