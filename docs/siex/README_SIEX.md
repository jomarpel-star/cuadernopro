# Preparación SIEX/CUE

Esta carpeta recoge la primera documentación interna de CuadernoPro para preparar una futura exportación asistida SIEX/CUE.

CuadernoPro no pretende, en esta fase, actuar como CUE comercial conectado directamente ni como aplicación integrada con el Interfaz Único. El objetivo inmediato es mantener el cuaderno local ordenado, revisar campos incompletos y generar información estructurada para que pueda ser revisada y tramitada por el agricultor, asesor o entidad autorizada.

CuadernoPro pretende:

- Ayudar al agricultor o asesor a mantener datos ordenados.
- Revisar campos incompletos antes de preparar una exportación.
- Generar exportaciones estructuradas o paquetes de trabajo.
- Preparar documentación para su posterior tramitación en sistemas oficiales.

> CuadernoPro no sustituye el sistema oficial SIEX/CUE ni realiza envíos telemáticos. La presentación o carga oficial de datos deberá realizarla el agricultor, asesor o entidad autorizada mediante los canales oficiales y con su certificado digital o mecanismo de identificación correspondiente.

Esta documentación no implementa conexión externa, llamadas HTTP, autenticación, certificados, envío telemático de datos ni importación desde administraciones.

## Decisión actual de producto

CuadernoPro seguirá siendo una herramienta local de gestión del cuaderno de explotación. Su objetivo respecto a SIEX/CUE será preparar y revisar datos para exportación asistida. La carga oficial, firma, presentación o comunicación con sistemas administrativos quedará en manos del agricultor, asesor o entidad autorizada.

Una conexión directa futura solo se consideraría si algún día compensa técnica y comercialmente, y siempre dependería de documentación técnica oficial, autorización del titular, requisitos autonómicos, certificados, seguridad y entorno de pruebas.

## Referencia Murcia

Para Murcia, el FEGA lista como referencia:

- Acceso al Interfaz Único: https://sede.carm.es/IUWS
- Registro de Explotaciones Agrarias de la Comunidad Autónoma de la Región de Murcia
- Portal de Entidades Colaboradoras
- Contacto: cuaderno_digital_agri@carm.es

Esta información se mantiene como referencia administrativa para confirmar formatos admitidos, plantillas, vías de carga o requisitos autonómicos. No debe usarse en código ni asumirse como especificación técnica de integración.
