CuadernoPro

El cuaderno agrícola libre para que ningún agricultor se quede sin herramienta.

Descargar para Windows Ver primeros pasos Ver código fuente

Gratis y completo

El programa completo está disponible para agricultores sin demo capada.

Sin cuotas ni activación

No exige licencias anuales obligatorias ni activación para funcionar.

Tus datos son tuyos

La base de datos y los documentos quedan bajo control del usuario.

PDF, SIEX y backups

Permite generar PDF del cuaderno, revisar SIEX y crear copias de seguridad.

Software libre

El código se publica bajo GPLv3 o posterior y la documentación bajo CC BY-SA.

Confianza y datos locales

  • Funciona en local.
  • Los datos se guardan en el ordenador del agricultor.
  • No requiere Docker ni WSL en la versión instalable de Windows.
