  • SHA256SUMS.txt: https://github.com/jomarpel-star/cuadernopro/releases/download/v8.4.7/SHA256SUMS.txt
  • Código fuente: https://github.com/jomarpel-star/cuadernopro

Cómo instalar

  1. Descarga el instalador Windows desde la release recomendada.
  2. Ejecuta el archivo CuadernoPro-8.4.7-Setup.exe.
  3. Sigue el asistente de instalación.
  4. Abre CuadernoPro desde el acceso directo creado.

No necesitas Docker, WSL, Python ni terminal para usar la versión instalable de Windows.

Comprobar SHA256

La comprobación SHA256 es opcional, pero recomendable para verificar que el archivo descargado coincide con el publicado.

certutil -hashfile CuadernoPro-8.4.7-Setup.exe SHA256

Compara el resultado con el archivo SHA256SUMS.txt publicado en la release de GitHub.

Aviso de Windows al instalar

Windows puede mostrar un aviso de seguridad al descargar o ejecutar CuadernoPro porque el instalador todavía no está firmado digitalmente y el proyecto es reciente.

Este aviso no significa necesariamente que el programa sea peligroso. Indica que Windows no reconoce aún al editor o que el archivo no tiene reputación suficiente.

Para instalar con seguridad:

  1. Descarga CuadernoPro solo desde esta página oficial o desde GitHub Releases.
  2. Comprueba que el archivo se llama CuadernoPro-8.4.7-Setup.exe.
  3. Verifica el SHA256 publicado en la release.
  4. Si confías en el origen, en el aviso de Windows pulsa Más información y después Ejecutar de todas formas.

Estamos estudiando la firma digital del instalador para futuras versiones.

Dónde se guardan los datos

En Windows instalado, CuadernoPro guarda la base de datos, documentos, copias de seguridad, exportaciones y datos de trabajo en:

Documentos\CuadernoPro

Desinstalar sin borrar datos

Desinstalar el programa no debe usarse como forma de borrar datos. Antes de desinstalar, crea una copia de seguridad desde CuadernoPro y conserva la carpeta Documentos\CuadernoPro.

Si quieres retirar la aplicación y mantener el historial, no borres esa carpeta.

Importar catálogos SIEX

Si vas a usar la revisión SIEX, descarga el ZIP oficial de catálogos que corresponda, abre la sección de catálogos SIEX en CuadernoPro, selecciona el ZIP e importa los datos. Revisa el diagnóstico al terminar.
