Reportar errores en GitHub

Al reportar un error, incluye la versión de CuadernoPro, sistema operativo, pasos para reproducir el problema y el mensaje de error si aparece.

No adjuntes bases de datos, facturas, recetas ni documentos con datos reales en issues públicos.
