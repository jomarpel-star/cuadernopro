Privacidad

CuadernoPro funciona en local y deja los datos bajo control del usuario.

Datos locales

CuadernoPro guarda los datos en el ordenador del usuario. En Windows instalado, la carpeta habitual es:

Documentos\CuadernoPro

La base de datos, documentos, copias de seguridad y exportaciones permanecen bajo control del usuario.

Sin subida automática

CuadernoPro no sube datos automáticamente a servidores externos.

Control del usuario

El usuario decide qué datos introduce, qué documentos adjunta, qué copias de seguridad conserva y qué archivos comparte con terceros.

Copias de seguridad

El usuario debe hacer copias de seguridad suficientes y conservarlas en una ubicación segura. Es recomendable guardar alguna copia fuera del equipo principal.

Descargas y soporte externo

Si usas GitHub para descargar CuadernoPro o reportar errores, aplican las políticas de privacidad de GitHub.

Si contactas para soporte, comparte solo los datos necesarios para resolver la consulta. No envíes bases de datos, facturas, recetas ni documentos reales si no es imprescindible.

Cookies

La web pública de CuadernoPro no usa cookies propias de análisis, publicidad o seguimiento. Consulta la política de cookies para más información.
