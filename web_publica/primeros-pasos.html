Primeros pasos

Una ruta sencilla para empezar con el cuaderno agrícola desde cero.

  1. Instalar CuadernoPro.
    Descarga el instalador Windows desde la release recomendada y completa el asistente.
  2. Abrir la aplicación.
    Usa el acceso directo creado por el instalador. La aplicación se abre en el navegador del equipo.
  3. Importar catálogos SIEX.
    Si vas a usar revisión SIEX, importa el ZIP oficial de catálogos y revisa el diagnóstico.
  4. Crear explotación.
    Introduce los datos de la explotación, titular, registros y datos de contacto que correspondan.
  5. Crear campaña.
    Da de alta la campaña agrícola con sus fechas de inicio y fin.
  6. Crear parcelas.
    Añade las parcelas y referencias SIGPAC necesarias para tu explotación.
  7. Crear cultivos.
    Asocia cada cultivo a sus parcelas. Si procede, añade el marco de plantación para calcular árboles.
  8. Registrar tratamientos, fertilización, prácticas y cosecha.
    Anota cada actuación con fecha, cultivo, parcelas, productos, dosis, aplicador y observaciones necesarias.
  9. Generar PDF del cuaderno.
    Revisa la información y genera el PDF del cuaderno cuando necesites conservarlo o compartirlo.
  10. Hacer copia de seguridad.
    Crea un backup desde CuadernoPro y guarda el ZIP fuera del equipo principal siempre que puedas.
