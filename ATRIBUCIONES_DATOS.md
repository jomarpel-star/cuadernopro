# Atribuciones de datos externos

CuadernoPro puede trabajar con referencias y servicios de datos externos. Esta
aplicacion no es una aplicacion oficial de la administracion.

Los datos externos pertenecen a sus respectivas fuentes y deben utilizarse
respetando sus condiciones de reutilizacion, licencia y atribucion.

Antes de una distribucion publica o comercial hay que verificar los textos
exactos de atribucion exigidos por cada fuente.

## Fuentes a revisar

| Fuente o servicio | Posible uso | Estado de atribucion |
| --- | --- | --- |
| SIGPAC / FEGA | Referencias de parcelas, recintos y datos SIGPAC | Pendiente de verificar |
| PNOA / IGN, si se usa | Ortofotografia o capas cartograficas | Pendiente de verificar |
| OpenStreetMap, si se usa | Cartografia base o visualizacion de mapas | Pendiente de verificar |
| Otros servicios cartograficos | Capas base, mapas, servicios WMS/WFS u otros datos geograficos | Pendiente de verificar |

## Aviso

CuadernoPro no debe presentarse como fuente oficial de datos administrativos,
cartograficos o agrarios. Si se muestran o descargan datos externos, el usuario
debe comprobar su origen, actualidad, condiciones de uso y validez para el fin
previsto.
