# Marca CuadernoPro

CuadernoPro es software libre. El código puede usarse, estudiarse, modificarse
y redistribuirse conforme a la licencia GPLv3 o posterior.

La marca CuadernoPro, el nombre del proyecto, el logotipo y otros elementos
identificativos quedan controlados por el proyecto para evitar confusion entre
agricultores, asesores y entidades que usen la aplicación.

## Versiones modificadas

Se pueden crear versiones modificadas de CuadernoPro respetando la licencia del
código. Esas versiones no deben presentarse como versión oficial de CuadernoPro
si no cuentan con autorización expresa del proyecto.

Una versión modificada debe dejar claro, cuando proceda:

- que se basa en CuadernoPro;
- que contiene cambios propios;
- quien mantiene esa versión;
- que no es la versión oficial si no está autorizada.

## Usos permitidos

Se permite mencionar CuadernoPro de forma honesta, por ejemplo para indicar
compatibilidad, origen del código o derivacion:

- "Basado en CuadernoPro";
- "Compatible con datos de CuadernoPro";
- "Fork no oficial de CuadernoPro".

Ese uso no debe sugerir patrocinio, certificación, soporte oficial o autoría del
proyecto original si no existe.

## Motivo

El objetivo no es limitar la libertad del software. El objetivo es proteger a
los usuarios frente a versiones alteradas que puedan confundirse con la versión
oficial, especialmente cuando gestionan información agrícola, documentos,
copias de seguridad o procesos administrativos.
